# 灵伴 (LingBan)

AI 智能体应用，支持创建自定义智能体并与多种大模型对话。

## 项目结构

- `apps/server` - 后端 API 服务（Node.js + Express + PostgreSQL）
- `apps/mobile` - 移动端 APP（React Native + Expo）

## 技术栈

- 后端：Node.js, Express, TypeScript, Prisma, PostgreSQL
- 前端：React Native, Expo, TypeScript

## 开发

```bash
# 安装依赖
npm install

# 启动后端
npm run dev:server
```
