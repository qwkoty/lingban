# 灵伴 — 产品需求文档 (PRD)

## 1. 产品概述

灵伴是一款纯网页端的 AI 智能体应用,用户匿名登录即可使用,支持自定义智能体(头像、名字、人设、温度、最大 token、API key),并选择智能体进行多轮对话,对话记录持久保留。

- **目标用户**:希望快速创建和使用个性化 AI 智能体的普通用户
- **核心价值**:零门槛匿名使用、高度自定义智能体、流畅的对话体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 匿名用户 | 自动分配递增 ID | 创建/编辑/删除智能体、与智能体对话、查看对话历史 |

### 2.2 功能模块

1. **匿名登录**:首次访问自动创建用户,第一个用户 ID 为 1,后续递增
2. **智能体管理**:创建、编辑、删除智能体,配置头像/名字/人设/温度/最大 token/API key
3. **对话**:选择智能体开始对话,支持多轮对话,历史记录持久化
4. **个人中心**:查看用户 ID、昵称,编辑个人信息

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 我的 | 用户信息卡片 | 展示头像、昵称、用户 ID,支持编辑昵称 |
| 我的 | 设置列表 | 主题色选择、关于信息等 |
| 智能体 | 智能体列表 | 展示所有智能体卡片(头像/名称/模型/状态),支持编辑/删除 |
| 智能体 | 创建/编辑智能体 | 头像上传、名称、人设、模型提供商选择、温度、最大 token、API key |
| 对话 | 智能体选择 | 选择已有智能体开始对话 |
| 对话 | 聊天界面 | 消息气泡、输入框、流式回复、历史记录加载 |

## 3. 核心流程

### 3.1 匿名登录流程

用户打开网页 → 前端检查本地 token → 无 token 则请求后端创建用户 → 后端分配递增 ID 并返回 token → 前端存储 token → 进入应用

### 3.2 智能体创建流程

点击创建按钮 → 进入编辑页 → 上传头像/填写名称/人设/选择模型/配置温度和 token/填写 API key → 保存 → 返回列表

### 3.3 对话流程

选择智能体 → 加载历史记录 → 输入消息 → 后端调用 LLM API → 流式返回 → 保存对话记录

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**:深色背景(接近黑灰,#0a0a0f),非纯黑
- **渐变**:七彩极光渐变(蓝→青→绿→黄→橙→粉→紫),柔和通透,非刺眼霓虹
- **材质**:毛玻璃拟态卡片(半透明背景、细边框、轻阴影、大圆角、浅层模糊)
- **字体**:简洁现代无衬线字体,层级分明
- **布局**:移动端优先,底部悬浮毛玻璃导航栏
- **动效**:页面平滑过渡、卡片弹性缩放、渐变流动、列表逐个淡入上滑、呼吸动画、骨架屏脉冲

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 我的 | 用户信息卡片 | 毛玻璃卡片,头像+昵称+ID,彩虹渐变光晕背景 |
| 我的 | 设置列表 | 极简列表项,流光色带主题选择器,轻盈开关 |
| 智能体 | 智能体列表 | 悬浮玻璃卡片,头像+名称+模型,右下角彩虹渐变创建按钮 |
| 智能体 | 创建/编辑 | 顶部大头像预览,胶囊按钮选模型,渐变描边选中态,底部固定渐变按钮 |
| 对话 | 智能体选择 | 网格/列表展示智能体,点击进入对话 |
| 对话 | 聊天界面 | 消息气泡(用户右/智能体左),底部输入栏,流式打字效果 |
| 全局 | 底部导航 | 悬浮毛玻璃栏,选中项渐变高亮,滑块动画 |

### 4.3 响应式

- 移动端优先设计,最大宽度 480px 居中
- 桌面端自适应居中显示,两侧留白
- 触摸优化:大点击区域,弹性反馈

### 4.4 视觉关键词

Minimal UI, Rainbow Gradient, Glassmorphism, Premium, Smooth Motion, Micro Interaction, Floating UI, Soft Glow, Elegant, Futuristic, Clean Layout, Fluid Animation, Aurora Color Palette
